# 🐦 鸟类声音可视化器

一个现代化的网页应用，将鸟类声音转化为动态视觉艺术。使用Web Audio API和Canvas技术实现实时音频可视化。

## ✨ 功能特点

### 🎵 **音频处理**
- 支持上传MP3、WAV、OGG、M4A等音频格式
- 拖放上传功能
- 6种预设鸟类声音（夜莺、麻雀、老鹰、猫头鹰、鹦鹉、知更鸟）
- 实时音量控制

### 🎨 **可视化模式**
1. **波形图** - 传统的音频波形显示
2. **频谱图** - 频率分布条形图
3. **环形图** - 圆形频率分布图
4. **粒子系统** - 动态粒子效果
5. **全部显示** - 同时展示所有可视化效果

### 📊 **音频分析**
- 实时频率显示
- 音量分贝监测
- 音调识别（高音/中音/低音）
- 节奏检测（BPM）
- 32段频率分布图

### 🐦 **鸟类信息**
- 每种鸟类的详细信息
- 大小、重量、分布区域
- 声音特点描述
- 精美的图标展示

## 🚀 快速开始

### 在线使用
直接打开 `index.html` 文件即可使用，无需服务器配置。

### 本地运行
```bash
# 1. 下载项目
git clone [项目地址]

# 2. 进入项目目录
cd bird-sound-visualizer

# 3. 使用本地服务器运行（推荐）
# 使用Python
python3 -m http.server 8000

# 或使用Node.js
npx serve

# 4. 在浏览器中访问
# http://localhost:8000
```

### 部署到服务器
将整个项目文件夹上传到你的Web服务器即可。

## 🎯 使用指南

### 1. 上传音频
- 点击"选择音频文件"按钮上传本地文件
- 或将文件拖放到上传区域
- 支持所有常见音频格式

### 2. 使用预设声音
- 点击鸟类图标按钮加载预设声音
- 每种鸟类都有独特的可视化效果
- 查看右侧的鸟类详细信息

### 3. 控制播放
- 播放/暂停：控制音频播放
- 停止：停止播放并重置
- 音量：实时调整音量大小

### 4. 切换可视化模式
- 点击底部可视化模式按钮
- 每种模式有不同的视觉效果
- "全部显示"模式同时展示所有效果

## 🔧 开发指南

### 添加新的鸟类声音
1. 在 `visualizer.js` 的 `birdData` 对象中添加新鸟类：
```javascript
const birdData = {
    newBird: {
        name: "新鸟类",
        description: "描述",
        size: "大小",
        weight: "重量",
        habitat: "分布区域",
        soundChar: "声音特点"
    }
}
```

2. 在HTML中添加对应的按钮：
```html
<button class="sound-btn" onclick="loadPresetSound('newBird')">
    <i class="fas fa-icon"></i>
    <span>新鸟类</span>
</button>
```

## 🐛 常见问题

### Q: 为什么没有声音？
A: 检查浏览器是否支持Web Audio API，确保没有静音或音量过低。

### Q: 可视化效果不显示？
A: 确保音频文件已正确加载，检查浏览器控制台是否有错误。

### Q: 如何添加自己的鸟类声音？
A: 参考"开发指南"部分，或直接将音频文件拖放到上传区域。

### Q: 支持哪些浏览器？
A: 支持所有现代浏览器（Chrome 60+、Firefox 55+、Safari 11+、Edge 79+）。
