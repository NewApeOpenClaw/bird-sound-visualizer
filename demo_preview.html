鸟类声音可视化器演示

将鸟类声音转化为动态视觉艺术

🎬 实时演示预览

以下展示了鸟类声音可视化器的4种核心可视化模式。实际应用中，这些效果会根据音频的实时频率和音量动态变化。

运行完整演示

波形图模式

显示音频的实时波形，线条高度随音量变化，颜色随频率变化。

特点：传统音频显示方式，直观展示声音的振幅变化。

频谱图模式

显示频率分布，不同高度的条形代表不同频率段的强度。

特点：科学分析音频频率组成，颜色渐变增强视觉效果。

环形图模式

圆形频率分布，线条从圆心向外延伸，长度表示频率强度。

特点：艺术化展示音频数据，适合音乐可视化。

粒子系统模式

动态粒子效果，粒子位置和大小随音频频率和音量变化。

特点：最具创意的可视化方式，粒子动画增强沉浸感。

✨ 核心功能

音频上传

支持拖放上传MP3、WAV、OGG、M4A格式，6种预设鸟类声音。

实时分析

显示频率、音量、音调、节奏，32段频率分布可视化。

多模式可视化

4种可视化模式，支持"全部显示"同时展示所有效果。

鸟类信息

详细的鸟类资料：大小、重量、分布区域、声音特点。

响应式设计

支持桌面、平板、手机，现代化的深色主题界面。

纯前端技术

使用Web Audio API + Canvas，无需后端服务器。

🛠️ 技术实现

Web Audio API：实时音频分析和处理
HTML5 Canvas：高性能图形渲染
JavaScript：350+行核心逻辑代码
CSS3：现代化响应式设计
